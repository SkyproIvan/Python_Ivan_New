1. Обзор проекта
Проект использует язык Python. pytest для написания и запуска автотестов. Для взаимодействия с веб-страницами применяется Page Object, а для генерации красивых отчетов — инструмент Allure.
Требования и установка зависимостей.Перед началом работы убедитесь, что у вас установлен Python (версии 3.7 или выше). Установите менеджер пакетов pip, если он отсутствует.
Установите все необходимые библиотеки из файла requirements.txt = pip install -r requirements.txt
Эта команда pytest --alluredir=allure-results выполнит все тесты в проекте и сохранит результаты для отчета в папку allure-results
С помощью установленного Allure можно смотретьо отчет в удобном виде. Его нужно сгенерировать из полученных данных allure serve allure-results
Примечание: Если при запуске тестов вы видите ошибку fixture 'driver' not found, убедитесь, что вы запускаете команду pytest из корневой директории проекта, где расположен файл conftest.py.
